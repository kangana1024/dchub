# Backend Express Typescript

## Build And Run

Build :

```bash
$ yarn build
```

OR

```bash
$ npm run build
```

Run Develop :

```bash
$ yarn dev
```

OR

```bash
$ npm run dev
```

Run Production :

```bash
$ yarn start
```

OR

```bash
$ npm run start
```
